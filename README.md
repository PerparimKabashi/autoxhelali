# 🏆 AUTO FOLIE XHELALI

**Premium Car Wrapping Visualizer** - Dizajn SUPERIOR për klient SUPERIOR

## ✨ Karakteristikat

- 🎨 **Dizajn Luxury** - Gold & Black theme, tipografi elegante
- 🖼️ **Before/After Slider** - Si FaceApp
- 🎯 **AI Visualization** - Nano Banana 2 (fal.ai)
- 📱 **Mobile Responsive** - Perfekt në çdo pajisje
- ⚡ **Fast** - Rezultate në sekonda

## 🚀 Instalimi

```bash
# 1. Instalo dependencies
npm install

# 2. Kopjo environment file
copy .env.example .env.local

# 3. Shto FAL_KEY në .env.local
notepad .env.local

# 4. Starto serverin
npm run dev
```

## 🔑 API Key

Merr FAL_KEY falas nga: https://fal.ai/dashboard/keys

## 📱 Mobile Access

```bash
npm run dev -- --hostname 0.0.0.0
```

Pastaj hap `http://[IP]:3000` në telefon.
